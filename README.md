
# The Cinnamon Modpack (TestModpackA)

This is a custom Minecraft modpack, using the Quilt modloader, for MC 1.21.1.

For more information, see https://cinnamonbunniy.github.io/modpack/


Add `/build/pack.toml` or `/build/index.toml` at the end of the url to see said files.


For Players, you'll want to download [this starter-instance zip](https://cinnamonbunniy.github.io/modpack/list/TestModpackA/TestModpackA.zip) , and import it into [Prism Launcher](https://prismlauncher.org/) , using [the instructions found at this Notion page](https://cinnamonbunniy.notion.site/TheCinnamonModpack-Player-Guide-332708c4953d80c9baecfabb76e3f9a0).
